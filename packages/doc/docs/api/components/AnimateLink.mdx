# `AnimateLink`

This component extends la-danze-ui [`NavLink`](api/components/NavLink.mdx) (chich extends `NavLink` of [react-router-dom](https://reactrouter.com/web/api/NavLink)). It provides additional props for animating nested routing (see [explanation](usage/animation/routing.md#nested-routing)).


## Example

```jsx
import { AnimateLink } from 'la-danze-ui';

function App() {
  return (
    ...
    <AnimateLink to="/" exact title="Home" />
    ...
  );
}
```
