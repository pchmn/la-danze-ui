# `FlexItem`

This component creates a `div` and permits adding css flex properties to this `div` with props. It should be used with [`FlexLayout`](api/components/FlexLayout.mdx).


## Example

```jsx
import { FlexLayout, FlexItem } from 'la-danze-ui';

function App() {
  return (
    <FlexLayout>
      <FlexItem />
      <FlexItem flexGrow={1} alignSelf="flex-end" />
      ...
    </FlexLayout>
  );
}
```
