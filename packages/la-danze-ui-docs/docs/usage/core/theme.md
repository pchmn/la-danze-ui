## Usage

To use it, just wrapp your app with `Theme` and that's it:

```tsx
import React from 'react';
import ReactDOM from 'react-dom';
import { Theme  } from 'la-danze-ui';

function App() {  
  return (
    <Theme>
      <div>my app</div>
    </Theme>
  );
}
```
