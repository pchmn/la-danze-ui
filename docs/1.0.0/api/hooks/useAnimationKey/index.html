useAnimationKey()#

This hook returns an unique AnimationKey.

It is only utile if you have nested routes in your app. It will help prevent parent route to re render when child route changes. See explanation.

Example#

import { useAnimationKey } from 'la-danze-ui';
...
function App() {
const animationKey = useAnimationKey();
...
}

key: number;
