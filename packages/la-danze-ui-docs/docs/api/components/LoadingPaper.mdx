# `LoadingPaper`

This component extends material-ui [`Paper`](https://next.material-ui.com/components/paper/#main-content). It adds a `pending` prop, to show a progress bar at the top of `<Paper>`.   
See [demo](https://next.material-ui.com/components/paper/#main-content).


## Example

```jsx
import { LoadingPaper } from 'la-danze-ui';

function App() {
  return (
    <LoadingPaper pending={true}>
      content
    </LoadingPaper>
  );
}
```
