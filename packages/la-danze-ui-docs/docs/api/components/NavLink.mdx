# `NavLink`

This component extends extends `NavLink` of [react-router-dom](https://reactrouter.com/web/api/NavLink). It provides additional props for preventing refresh of current route.


## Example

```jsx
import { NavLink } from 'la-danze-ui';

function App() {
  return (
    ...
    <NavLink to="/" exact title="Home" noRefresh />
    ...
  );
}
```
