# `useAnimationKey()`

This hook returns an unique [`AnimationKey`](#animationkey).

It is only utile if you have nested routes in your app. It will help prevent parent route to re render when child route changes. See [explanation](usage/animation/routing.md#nested-routing).


## Example

```tsx
import { useAnimationKey } from 'la-danze-ui';
...

function App() {
  const animationKey = useAnimationKey();
  ...
}
```

## `AnimationKey`

```tsx
interface AnimationKey {
  key: number;
  updateKey: () => void;
}
```
