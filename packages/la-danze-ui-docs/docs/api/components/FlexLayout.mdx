# `FlexLayout`

This component creates a `div` with `display: flex`, and permits customizing more css flex properties with props. It should be used with [`FlexItem`](api/components/FlexItem.mdx).


## Example

```jsx
import { FlexLayout, FlexItem } from 'la-danze-ui';

function App() {
  return (
    <FlexLayout flexDirection="column" flexWrap="wrap">
      <FlexItem />
      ...
    </FlexLayout>
  );
}
```
