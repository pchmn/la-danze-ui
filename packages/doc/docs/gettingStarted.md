La Danze UI is available as a npm package.

## Installation

```bash
# npm
npm install la-danze-ui

# yarn
yarn add la-danze-ui
```

### Peer dependencies

* [`material-ui`](https://next.material-ui.com/) (next)
  * @material-ui/core@next
  * @material-ui/icons@next
  * @emotion/react
