# `DrawerListItem`

This component extends `ListItem` component of [material-ui](https://next.material-ui.com/). It also provide props for routing. See props.

> Can't be used outside [`<Drawer>`](api/components/Drawer.mdx).

## Example

```jsx
import { List, ListItemIcon, ListItemText } from '@material-ui/core';
import { DrawerListItem } from 'la-danze-ui';

function App() {
  return (
    ...
    <DrawerListItem to="/" exact title="Home">
      <ListItemIcon>...</ListItemIcon>
      <ListItemText primary="Home" />
    </DrawerListItem>
    ...
  );
}
```
