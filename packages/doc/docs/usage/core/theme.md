## Usage

To use it, just wrapp your app with `LaDanzeTheme` and that's it:

```tsx
import React from 'react';
import ReactDOM from 'react-dom';
import { LaDanzeTheme  } from 'la-danze-ui';

function App() {  
  return (
    <LaDanzeTheme>
      <div>my app</div>
    </LaDanzeTheme>
  );
}
```
